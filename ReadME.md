# README

## Project Overview

This is a portfolio-style website featuring smooth scrolling and creative animations. It uses [Locomotive Scroll](https://locomotivemtl.github.io/locomotive-scroll/) for smooth scrolling and [GSAP](https://greensock.com/gsap/) for animations.

## Structure

- `index.html` - Main HTML file containing the website structure.
- `style.css` - Custom styles for layout, fonts, and animations.
- `script.js` - JavaScript for scroll and GSAP animations.
- `domain.ttf`, `canopee.ttf` - Custom font files.

## Features

- Responsive design
- Custom fonts
- Smooth scrolling
- Animated transitions
- Portfolio sections

## Setup

1. Clone or download the repository.
2. Make sure all files (`index.html`, `style.css`, `script.js`, font files) are in the same directory.
3. Open `index.html` in your browser.

## Dependencies

- [Locomotive Scroll](https://cdn.jsdelivr.net/npm/locomotive-scroll@3.5.4/dist/locomotive-scroll.js)
- [GSAP](https://cdnjs.cloudflare.com/ajax/libs/gsap/3.11.5/gsap.min.js)
- [Remixicon](https://cdn.jsdelivr.net/npm/remixicon@3.2.0/fonts/remixicon.css)

## Customization

- Update images and text in `index.html` to personalize your portfolio.
- Modify styles in `style.css` for different themes or layouts.
